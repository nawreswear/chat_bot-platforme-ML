À propos de l'ISET Sfax

L'ISET Sfax a pour mission de former des cadres techniques hautement qualifiés capables de répondre aux besoins du marché du travail et de contribuer au développement économique et technologique de la Tunisie. Notre vision est de devenir un centre d'excellence en matière d'enseignement technologique et de recherche appliquée.

Présentation d'AutoML Pro

AutoML Pro est une plateforme d'apprentissage automatique développée par l'équipe du Laboratoire d'Intelligence Artificielle de l'ISET Sfax. Cette plateforme permet aux utilisateurs de créer, entraîner et déployer des modèles d'apprentissage automatique sans nécessiter de connaissances approfondies en programmation ou en science des données.

Caractéristiques principales

  • Interface intuitive : Une interface utilisateur conviviale qui guide l'utilisateur à travers le processus d'analyse de données et de création de modèles.
  • Prétraitement automatique : Détection et gestion automatique des valeurs manquantes, des valeurs aberrantes et normalisation des données.
  • Sélection de modèles : Évaluation automatique de plusieurs algorithmes d'apprentissage automatique pour trouver le plus performant pour vos données.
  • Visualisations interactives : Graphiques et visualisations pour mieux comprendre vos données et les performances des modèles.
  • Exportation des résultats : Possibilité d'exporter les modèles entraînés, les prédictions et les rapports d'analyse sous différents formats (CSV, PDF, etc.).
  • Déploiement facile : Intégration simplifiée des modèles dans vos applications et systèmes existants.

Types d'analyses supportées

AutoML Pro prend en charge trois types principaux d'analyses :

Type d'analyse Description Cas d'utilisation
Classification Prédire une catégorie ou une classe Détection de fraude, diagnostic médical, classification d'images
Régression Prédire une valeur numérique Prévision des ventes, estimation de prix, prédiction de consommation
Clustering Regrouper des données similaires Segmentation client, analyse de marché, détection d'anomalies

Guide d'utilisation

Étape 1 : Préparation des données

Avant de commencer, assurez-vous que vos données sont bien structurées et dans un format compatible (CSV, Excel ou JSON). Idéalement, vos données devraient être organisées avec :

  • Une ligne d'en-tête contenant les noms des colonnes
  • Chaque ligne représentant une observation ou un échantillon
  • Des valeurs numériques pour les analyses quantitatives
  • Des catégories clairement définies pour les analyses de classification

Étape 2 : Chargement des données

  1. Accédez à l'interface principale d'AutoML Pro
  2. Sélectionnez le type d'analyse que vous souhaitez effectuer (Classification, Régression ou Clustering)
  3. Cliquez sur le bouton de téléchargement pour sélectionner votre fichier de données
  4. Une fois le fichier chargé, vous pourrez prévisualiser vos données pour vérifier qu'elles sont correctement importées

Étape 3 : Configuration de l'analyse

Pour les analyses supervisées (Classification et Régression) :

  1. Sélectionnez la colonne cible (la variable que vous souhaitez prédire)
  2. Configurez les options de prétraitement selon vos besoins
  3. Sélectionnez les modèles que vous souhaitez évaluer
  4. Définissez les paramètres d'évaluation (taille de l'ensemble de test, validation croisée, etc.)

Pour les analyses non supervisées (Clustering) :

  1. Sélectionnez les colonnes à inclure dans l'analyse
  2. Choisissez l'algorithme de clustering
  3. Définissez le nombre de clusters (pour K-Means) ou les paramètres spécifiques à l'algorithme

Étape 4 : Exécution de l'analyse

Cliquez sur le bouton "Lancer l'analyse" pour démarrer le processus. AutoML Pro va :

  1. Prétraiter vos données selon les options sélectionnées
  2. Diviser les données en ensembles d'entraînement et de test (pour les analyses supervisées)
  3. Entraîner et évaluer les modèles sélectionnés
  4. Générer des visualisations et des métriques de performance

Étape 5 : Interprétation des résultats

Une fois l'analyse terminée, vous pourrez explorer les résultats à travers différents onglets :

  • Résumé : Vue d'ensemble des performances des modèles et du meilleur modèle
  • Données : Exploration des données et statistiques descriptives
  • Modèles : Comparaison détaillée des performances des différents modèles
  • Visualisations : Graphiques et visualisations pour mieux comprendre vos données et les résultats

Étape 6 : Exportation et déploiement

Vous pouvez exporter les résultats de votre analyse sous différentes formes :

  • Téléchargement du modèle entraîné pour une utilisation ultérieure
  • Exportation des prédictions au format CSV
  • Génération d'un rapport PDF détaillant l'analyse et les résultats

Options avancées

Prétraitement des données

AutoML Pro offre plusieurs options avancées pour le prétraitement des données :

  • Gestion des valeurs manquantes : Imputation par la moyenne, la médiane, le mode ou suppression des lignes
  • Détection des valeurs aberrantes : Identification et traitement des outliers par différentes méthodes statistiques
  • Normalisation des données : Standardisation, Min-Max scaling, ou normalisation robuste
  • Encodage des variables catégorielles : One-hot encoding, label encoding, ou target encoding

Paramètres des modèles

Pour les utilisateurs avancés, AutoML Pro permet de configurer les hyperparamètres des modèles :

  • Classification : Paramètres spécifiques pour la régression logistique, Random Forest, SVM et XGBoost
  • Régression : Configuration des modèles de régression linéaire, Ridge, Lasso et Random Forest
  • Clustering : Paramètres pour K-Means, clustering hiérarchique et DBSCAN

Validation croisée

La validation croisée est une technique importante pour évaluer la performance des modèles. AutoML Pro permet de configurer :

  • Le nombre de folds (partitions) pour la validation croisée
  • La stratégie de validation (stratifiée pour les problèmes de classification déséquilibrés)
  • Les métriques d'évaluation spécifiques à utiliser

FAQ

Questions fréquemment posées

Quels formats de fichiers sont supportés ?

AutoML Pro prend en charge les formats CSV (.csv), Excel (.xls, .xlsx) et JSON (.json).

Quelle taille de données puis-je analyser ?

La version standard d'AutoML Pro peut traiter des ensembles de données jusqu'à 100 Mo. Pour des ensembles plus volumineux, veuillez contacter notre équipe pour une solution adaptée.

Comment puis-je interpréter les résultats de mon modèle ?

AutoML Pro fournit des visualisations et des métriques pour vous aider à comprendre les performances de votre modèle. Pour la classification, consultez la matrice de confusion et les courbes ROC. Pour la régression, examinez les graphiques de dispersion et les métriques d'erreur.

Puis-je utiliser mes propres algorithmes ?

La version actuelle d'AutoML Pro ne permet pas d'intégrer des algorithmes personnalisés. Cette fonctionnalité est prévue pour une version future.

Comment puis-je déployer mon modèle en production ?

Vous pouvez exporter votre modèle entraîné et l'intégrer dans votre application. Pour une assistance sur le déploiement, veuillez consulter notre équipe technique.
