Guide d'utilisation
Comment utiliser AutoML Pro
  1. Sélectionnez le type d'analyse que vous souhaitez effectuer
  2. Téléchargez votre fichier de données (CSV, Excel ou JSON)
  3. Sélectionnez la colonne cible pour les tâches supervisées
  4. Configurez les options avancées si nécessaire
  5. Cliquez sur "Lancer l'analyse" pour démarrer
Types d'analyses disponibles
  • Classification - Prédire une catégorie
  • Régression - Prédire une valeur numérique
  • Clustering - Regrouper des données similaires
Conseils pour de meilleurs résultats
  • Assurez-vous que vos données sont propres et bien formatées
  • Utilisez un ensemble de données suffisamment grand
  • Vérifiez que votre colonne cible est correctement définie
  • Activez les options de prétraitement pour améliorer la qualité
  • Comparez plusieurs modèles pour trouver le plus performant
Formats de fichiers supportés
  • CSV (.csv) - Valeurs séparées par des virgules
  • Excel (.xls, .xlsx) - Feuilles de calcul Microsoft Excel
  • JSON (.json) - Format JavaScript Object Notation
